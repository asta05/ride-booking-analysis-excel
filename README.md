# RIDE BOOKING ANALYSIS (EXCEL DASHBOARD)

##  Objective

Analyzed ride booking data to identify cancellation trends, revenue insights, and key factors affecting ride completion.

##  Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Slicers

##  Key Analysis

* Calculated cancellation rate and revenue metrics
* Identified major reasons for cancellations (no-shows, driver unavailability)
* Analyzed booking patterns based on time and location

##  Dashboard Features

* Interactive dashboard using Pivot Tables and Slicers
* Visual representation of KPIs (cancellation rate, revenue)
* Easy filtering for better insights

## Files Included

* `ride_booking_analysis.xlsx` – Excel dashboard and data
* `dashboard.png` – Dashboard preview
